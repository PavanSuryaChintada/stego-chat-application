## [1.0.0] - 2025-01-16

### Added
- Initial release of StegoChat
- End-to-end encrypted messaging
- LSB steganography for hiding messages in images
- Real-time chat functionality using Supabase
- User authentication (sign up/sign in)
- User search and chat creation
- Modern UI with dark theme
- Passcode protection for steganographic messages
- Message checksum verification
- Automatic message decryption
- Toast notifications for user feedback
- Responsive design for all devices

### Security
- AES-GCM encryption for messages
- PBKDF2 key derivation
- Random salt and IV generation
- Row Level Security (RLS) in Supabase
- Secure image storage with Supabase Storage

### Tech Stack
- React 18.3 with TypeScript
- Vite 5.4 for build tooling
- Supabase for backend services
- TailwindCSS for styling
- shadcn/ui for UI components
- Lucide React for icons
